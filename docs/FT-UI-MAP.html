Every input, every dropdown, every table column, every button traces back to a specific FT parameter, API endpoint, or CLI argument. No exceptions.

Data source: FT REST API endpoints (40+) for live data. FT CLI for backtesting/hyperopt/data. FT config.json for settings. Strategy .py files for code.
